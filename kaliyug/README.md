# Kaliyug — Project Package (docs)

The script, Movie Bible, art direction, trailer screenplays, and reference docs for **Kaliyug** —
a Soulsborne-grade dark-mythic game/trailer (Bloodborne / Elden Ring register) fusing Victorian-Gothic
+ Lovecraftian cosmic horror with **Indo-Gothic** Indian architecture; the gods **reimagined** in
aesthetic but kept **recognizable**. Images are intentionally excluded — **generate from the prompt
list** (see below).

## Where the generation prompts live
- **`../data/worklist.json`** — the ready-to-generate batch: **47 look-locked prompts** across all 17
  beats of the re-cut trailer (each: `id`, 5-layer `positive`, `negative`, `aspect` 16:9,
  `save_as: frame_<id>.png`). This is the Codex/generation worker's input.
- Direkta projects: **`LSKaw2WE4E`** = the current re-cut (v3, new art direction); `QdReqp5q7c` = the
  earlier v2 cut (kept as a record).

## Hard canon (do not break)
Grounded mythic, **never tech/AI/cyberpunk/Vyom**. **Kali** = demon of the age, **never Goddess Kali**,
**abstract from the start** (empty throne / smoke-thought / whisper). **Sacred geometry only in the
dream beat.** Corruption-horror is **Kali's alone**; the deities stay clean/numinous. Deities:
reimagine the *style*, keep the *identity* (negate calendar-art, never the god).
