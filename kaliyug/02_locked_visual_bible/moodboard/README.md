# KALIYUG — Moodboard

Visual reference for **`../kaliyug_game_art_direction.md`** (the project-wide art direction).
Drop reference images into each category subfolder; the `references.md` in each lists the specific
pulls and *why* they matter for KALIYUG.

**Aesthetic in one line:** Soulsborne dread + Victorian-Gothic / Lovecraftian cosmic horror, fused
with Indian (Indo-Gothic) architecture — *"corrupt the yantra, don't import the tentacle."* The
deities are reimagined in **aesthetic** but stay **recognizable, shaped as the deity.**

## Hard rules (from the art direction)
- **Sacred geometry / cosmic imagery only in the dream refs** (`06`). The real-world refs (`01`–`05`, `07`) stay **grounded Indo-Gothic stone-and-rust**.
- **Corruption / decay / body-horror is Kali's alone** — the deities (`06`) stay **clean and numinous**.
- **Desaturated ground, one accent per frame** — crimson `#8B1A1A` *or* blue-white `#BFE3F2`, never both.
- Deities: **keep the form, reimagine the style** — negate calendar-art, never the deity's identity.
