# Kaliyug Concept Trailer — Codex Image Batch Handoff

## Purpose
This package gives Codex the project context, locked visual direction, world-first trailer structure, prompt bank, and quality rules required to generate batches of concept-trailer keyframes for sorting and selection.

## Important
- This ZIP intentionally contains **markdown files only**.
- Do **not** rely on prior chat context. Every prompt in this package is standalone.
- Do **not** attach or embed reference images inside generated outputs unless explicitly requested later.
- Generate images in **16:9 landscape**.
- Target **clean, high-detail, 4K-quality output**.
- The final trailer will be approximately **3 minutes**, built from world-first keyframes that can later be animated into 5-8 second video shots.
- The trailer should communicate the concept by showing the world, eras, corruption, sacred geography, and mythic scale. Characters are secondary unless a frame explicitly calls for them.

## Top-Level Locked Direction

**Kaliyug trailer visual style:** realistic fantasy. The real world is post-apocalyptic, desolate, frightening, gory, red-themed, and corrupted by demonic dark magic. Vein-like organic corruption branches through ruins, temples, roads, machines, and landscapes. Sacred geometry and cosmic metaphysical visuals appear only in dream realms, visions, and spiritual-revelation sequences. Kali is the demon/personification of Kali Yuga, never Goddess Kali. The trailer should feel grounded, mythic, brutal, spiritual, cinematic, and world-first.
