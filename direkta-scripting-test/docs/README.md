# docs/

Durable design + architecture docs for the Direkta scripting pipeline. The agent **rule
files** in [`../agents/`](../agents/) are the canonical source of truth; everything here is
the reasoning and the whole-system picture around them.

| File | What |
|---|---|
| [`pipeline.md`](pipeline.md) | Design rationale — why the 3-agent pipeline is built this way, per-agent mechanics, the Maurya Origins validation, the path to production, and open threads. |
| [`architecture.html`](architecture.html) | The whole-system visual architecture doc (stages, gates, agent leash, schema). Open in a browser. |

See also: [`../WORKFLOW_SUMMARY.md`](../WORKFLOW_SUMMARY.md) (prototype summary) and
[`../README.md`](../README.md) (how to run the harness).
