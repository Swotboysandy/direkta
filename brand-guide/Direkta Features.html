PROJECT MANIFEST · DIRECTOR-FACING APP

Every feature.
One list.

Direkta is built as seven workspaces sharing one database, each purpose-built for one phase of pre-production. The pipeline runs Scripting → Casting → Storyboard → Stitch → Export, with Dashboard above and Library beside. Below is every feature shipped in the prototype to date.
